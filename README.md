# Healthcare EMR Dashboard

Spring Boot Healthcare EMR Dashboard project with REST APIs and Swagger documentation.

## Technologies Used
- Java
- Spring Boot
- REST API
- Swagger OpenAPI
- Maven

## API Endpoints
- GET /api/patients
- GET /api/patients/{id}
- POST /api/patients
- DELETE /api/patients/{id}

## Swagger URL
http://localhost:8080/swagger-ui/index.html
